
# Generelles

Du findest hier eine Liste von möglichen Alternativen zu den bisher üblicherweise genutzten Big Tech Diensten die vornehmlich aus den USA stammen. Die Alternativen sind am besten EU-Dienste, am besten Open Source und wichtig ist, dass Privatsphäre und Sicherheit bei den Alternativen ernst genommen werden. Allerdings ist allen hier gelisteten Alternativen gemein, dass sie die vorher genutzten Dienste auch "gut" ersetzen sollen, also z.B. wenn üblich eine Synchronisation über alle genutzten Geräte anbieten. Der Hauptfokus liegt nicht einfach bei maximaler Sicherheit, sondern eben auch auf Nutzbarkeit.

Bitte gerne mit eigenen Ideen ergänzen! In Klammern gerne Begründungen hinter die Alternative schreiben. Fragezeichen stehen für noch nicht selbst ausprobierte Ideen.

Inspirationen: [https://european-alternatives.eu](https://european-alternatives.eu), ...
